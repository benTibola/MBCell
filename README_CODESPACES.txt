BROWSER-ONLY ROUTE (NO DOCKER INSTALL ON YOUR PC)

1. Create a new GitHub repository in your browser.
2. Upload the CONTENTS of this folder to that repo.
3. On the repo page, click the green Code button.
4. Click Codespaces.
5. Click Create codespace on main.
6. Wait for the codespace to open in the browser.
7. In the terminal, run:
   bash run_b2_codespace.sh
8. When it finishes, download this file from the codespace file explorer:
   build/B2_public_run_01_min.zip
9. Upload that ZIP back to ChatGPT.
